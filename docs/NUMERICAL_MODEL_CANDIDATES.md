# Global Numerical Model Candidates For AI_SSTP

AI_KHOA_SSTP 후속 모델에서 표층 수온 예측 입력자료로 활용할 수 있는 전지구 수치모델 후보군을 정리한 문서입니다.

## 목적

- 단일 정점 LSTM 이후 단계에서 사용할 수 있는 전지구 해양 수치모델 후보 검토
- 서해 연안 SST 예측에 필요한 입력 변수 확보
- 향후 물리 기반 또는 physics-informed 모델 확장 시 사용할 외부 해양장 선정

## 선정 기준

- 전지구 또는 광역 해양 커버리지
- 표층 수온 포함
- 해류, 염분, 해수면고 등 보조 물리 변수 제공 여부
- 운영 예측 또는 재분석 자료 제공 여부
- 다운로드/API 접근성
- 문서화 수준과 활용 사례

## 후보군

### 1. Copernicus Marine Global Ocean Physics

- 대표 제품:
  - `GLOBAL_ANALYSISFORECAST_PHY_001_024`
- 특징:
  - 전지구 해양 물리 분석·예측 자료 제공
  - SST, 염분, 해류, 해수면고 등 다양한 물리 변수 확보 가능
  - 운영성과 문서화 수준이 높아 실무 적용이 쉬움
- 장점:
  - 데이터 접근성과 메타데이터 정리가 우수함
  - 예측과 분석 자료를 함께 활용하기 좋음
  - 후속 모델 입력 피처 확장에 유리함
- 활용 방향:
  - 1순위 후보
  - 표층 수온 + 표층 해류 + 해수면고 조합 입력에 적합
- 공식:
  - https://data.marine.copernicus.eu/product/GLOBAL_ANALYSISFORECAST_PHY_001_024/description

### 2. NOAA GOFS 3.1

- 기반:
  - HYCOM 계열 전지구 해양 예측장
- 특징:
  - 전지구 해양 예측 자료 제공
  - SST, 수온 연직 구조, 유속 등 확보 가능
- 장점:
  - 운영 자료 활용성이 높음
  - 해양 예측 입력자료로 널리 사용됨
- 활용 방향:
  - Copernicus와 함께 비교 실험용 2순위 후보
  - 서해 인근 격자 추출 후 정점 매칭에 적합
- 공식:
  - https://www.hycom.org/dataserver/gofs-3pt1/analysis

### 3. HYCOM Global

- 특징:
  - hybrid coordinate ocean model 기반 전지구 해양장
  - 다양한 운영 예측과 연구 자료의 기반 모델로 사용됨
- 장점:
  - 수온, 염분, 유속 등 핵심 해양 변수 확보 가능
  - GOFS와 계통적으로 연결되어 비교 검토가 쉬움
- 활용 방향:
  - 독립 후보라기보다 GOFS 계열 검토의 확장 축으로 적합
  - 해양 물리 피처 엔지니어링용 입력장으로 활용 가능
- 공식:
  - https://www.hycom.org/

### 4. ECCO

- 특징:
  - 전지구 해양 상태 추정 자료
  - 운영 예측보다는 물리 일관성이 높은 분석장에 강점
- 장점:
  - 물리 기반 해석과 retrospective 분석에 적합
  - 단순 성능 비교보다 물리적 설명력 확보에 유리
- 활용 방향:
  - physics-informed 모델 설계 또는 사후 검증용 후보
  - 실시간 운영보다는 연구/분석 보조 축으로 적합
- 공식:
  - https://ecco-group.org/

### 5. ECMWF / Copernicus Climate 계열

- 특징:
  - 대기 forcing 및 재분석 자료 활용성이 높음
  - 해양 단독 예측장보다 외생 기상 입력 측면에서 가치가 큼
- 장점:
  - 기온, 풍속, 기압, 열 플럭스 관련 변수 보강 가능
  - 기존 LSTM 입력 변수 확장과 자연스럽게 연결됨
- 활용 방향:
  - 해양 수치모델 단독 후보보다는 보조 forcing 자료로 적합
  - 대기-해양 결합 입력 구성에 유리
- 공식:
  - https://www.ecmwf.int/
  - https://cds.climate.copernicus.eu/

### 6. WW3 / NOAA Wave 계열

- 특징:
  - 파랑 및 해상 상태 입력자료 제공
  - 직접적인 SST 모델은 아니지만 파고, 바람, 해면 상태 보강에 유용
- 장점:
  - 기존 feature 중 유의파고와 직접 연결 가능
  - 연안 변동성 설명력 향상 가능
- 활용 방향:
  - 해양 물리장 주입력과 함께 쓰는 보조 입력 후보
- 공식:
  - https://polar.ncep.noaa.gov/waves/

## 국내 (대한민국) 연안 고해상도 수치모델 검토

한반도 주변 해역(특히 조석 격차가 크고 지형이 복잡한 서해안)을 타겟으로 할 때, 복잡한 지형에 특화된 국지 모델들이 존재합니다.
- **국립해양조사원(KHOA) 현업 해양예측시스템**: 서/남해에 특화된 초고해상도 운영 모델 기반의 조위, 조류, 수온 결합 예측장 제공
- **기상청(KMA) 국지 해양예측모델**: 기상예보장과 강하게 연동되어 대기 변동 예측 반영 우수
- **KIOST (한국해양과학기술원)**: 연구 목적 최상급 정확도의 해양 순환 예측 시스템 운영

### 국내 수치모델 도입의 현실적인 한계점 (데이터 파이프라인 접근성)

비록 연안 특화로 물리적 정확성은 우수하나, 예측 자동화 파이프라인 관점에서 **모델 예측 격자장(Grid data)에 대한 외부 자동 다운로드가 매우 어렵다**는 큰 제약이 있습니다.
- **오픈 API 부재**: OPeNDAP와 같이 Python 패키지로 원하는 격자망만 잘라서 실시간 자동 다운로드하는 환경이 대외적으로 열려있지 않습니다.
- **오프라인 연계 필요성**: 공공포털 등을 통해 개별 정점 단위 수치는 일정 기간 API 조회가 가능하지만, 전체 광역 모델 원본 파일(`.nc`) 확보는 보통 폐쇄망 내에서 운영되거나 기관 협력을 통한 비동기 다운로드/오프라인 송수신이 필수적입니다.

> **결론 및 제안**: 국내 데이터 연동 진입장벽이 높으므로 초기 개발 시 파이프라인이 즉시 통과되는 **글로벌 오픈 모델(Copernicus, NOAA)로 전체 자동 다운로드 뼈대를 구축**합니다. 본 시스템이 안정화된 후, 협력 기관(KHOA)으로부터 고해상도 원본 파일을 제공받으면 해당 데이터 구역의 파일 경로만 스왑(Swap)하여 적용하는 **투트랙(Two-track) 전략**을 채택해야 합니다.

## 추천 우선순위

### 1순위

- Copernicus Marine Global Ocean Physics
- 이유:
  - 접근성, 변수 구성, 문서화, 운영성이 가장 균형적

### 2순위

- NOAA GOFS 3.1 / HYCOM Global
- 이유:
  - 전지구 운영 자료로서 안정적이며 해양 변수 구성이 우수함

### 3순위

- ECCO
- 이유:
  - 운영 예측보다는 물리 기반 분석 및 설명력 확보에 유리함

## AI_SSTP 적용 제안

### 기본 입력 구성

- 현행 관측 입력:
  - 수온
  - 기온
  - 풍속
  - 기압
  - 유의파고

- 추가 검토할 수치모델 입력:
  - 표층 수온
  - 표층 해류 u/v
  - 해수면고
  - 표층 염분
  - 혼합층 관련 변수

### 추천 조합

- 주 입력:
  - Copernicus Marine Global Ocean Physics 또는 GOFS 3.1
- 보조 입력:
  - ECMWF/Copernicus 기상 forcing
  - WW3 파랑 자료
